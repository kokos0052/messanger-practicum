## Описание

Проект написан на TypesScript, для упрощенной шаблонизации используется кастомная JSX фабрика и SCSS.

Роутинг основан на кастомном решении с опорой на примере данного в матералах курса

Запросы используют XHR

Для сборки проекта используется Vite

Деплоится на netlify, ссылка:

https://unique-phoenix-f69024.netlify.app/

## Установка

- `npm install` — установка стабильной версии,
- `npm build` — сборка проекта,
- `npm start` — запуск приложения после сборки,
- `npm run dev` — запуск проекта в режиме разработки.
